## Overview
This project focuses on building an interactive **sales analytics dashboard** to evaluate the performance of an e-commerce business.  
The dashboard consolidates sales, profit, and quantity metrics into a single view, enabling quick insights and informed business decisions.

---

## Problem Statement
Raw sales data alone does not provide clear insights into:
- Which product categories perform best
- How profits vary across categories and sub-categories
- Which payment methods contribute most to revenue

This dashboard addresses these challenges through effective data visualization.

---

## Key Performance Indicators (KPIs)
- **Total Orders Quantity:** 5,615  
- **Overall Revenue:** 438K  
- **Net Profit:** 37K  

---

## Dashboard Features
- Category-level performance comparison  
- Sub-category profit contribution analysis  
- Payment mode impact on revenue and profit  
- Interactive category filters for focused analysis

## Insights Gained
- Electronics generated the highest revenue
- Clothing and Electronics contributed major portions of total profit
- Card-based payments showed stronger profit contribution compared to COD
- Category filters improved targeted analysis
